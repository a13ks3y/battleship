# Battleship

A premium, modern Battleship experience built with pure HTML, CSS, and vanilla JavaScript. No frameworks, no dependencies—just a clean, polished UI with smooth interactions and audio.

## Play the game
- Live demo: https://a13ks3y.github.io/battleship/

## Features
- Classic Battleship gameplay with player vs. computer
- Smart ship placement with required spacing/clearance
- Polished UI with animations and parallax background
- Clear hit/miss/sunk visual feedback
- Background music and sound effects with toggle

## How to play
1. Place your ships on the grid (rotate as needed).
2. Start the battle and click the enemy grid to attack.
3. First to sink all enemy ships wins.

## Controls
- Click to place or attack
- Rotate ships during placement (UI control)
- Toggle audio on/off

## Tech stack
- HTML5
- CSS3 (Grid, Flexbox, animations)
- Vanilla JavaScript (ES6+)

## Run locally
Just open `index.html` in your browser.

## Project goals
- Zero external dependencies
- Premium look-and-feel and smooth UX
- Clear, responsive feedback during gameplay
